# Data-Transformation-Power-Query

## Project Overview
This project demonstrates data cleaning and transformation using Power Query in Power BI. The objective is to prepare raw HR analytics data for analysis and dashboard creation by applying real-world data transformation techniques.

## Tools Used
- Power BI Desktop
- Power Query Editor

## Dataset
- HR Analytics Dataset (CSV format)

## Key Transformations
- Removed unnecessary columns
- Renamed columns for clarity
- Applied correct data types
- Created derived columns:
  - Age Group
  - Salary Band
- Prepared clean, analysis-ready data model

## Outcome
This project showcases practical data transformation skills and demonstrates the ability to prepare structured datasets for effective business analysis.
